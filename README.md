## Features
- Blazing-fast development with Next.js.
- Clean, modular architecture with components, models, config, and utils.
- Optimized font loading using `next/font`.
- Ready for deployment on platforms like Vercel.

## Prerequisites
- Node.js v16+
- npm or yarn
